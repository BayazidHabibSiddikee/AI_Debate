# AI Debate & Research Playground

A multi-agent chat playground designed to generate synthetic conversation datasets. 

## Mission
Our goal is to simulate deep, multi-character debates to brainstorm better ideas through AI-driven discussion and research. Eventually, the exported datasets (`storage/debates_dataset.jsonl`) generated by this playground can be used to train action mappers (mapping nuanced texts to complex physical animations) for VRM models.

## Project Architecture
This project is cleanly split into two distinct processes:

1. **The Debate Simulator** (`app.py` on Port 5050): The core Flask and LangChain application that orchestrates the AI debates and exports the conversational turns.
2. **The Research & RAG Engine** (`rag_server.py` on Port 5080): Provides RAG (Retrieval-Augmented Generation) processing, news harvesting, and document context. The debate simulator optionally connects to this to ground characters with real research data.

## 1. Setting up the Debate Simulator

### Installation
The simulator uses a lightweight dependency list.
```bash
pip install -r requirements-debate.txt
```

### Environment Variables
Create a `.env` file in the root directory. You can use local models (like Ollama or LM Studio) or OpenAI.
```env
OPENAI_API_KEY="not-needed-for-local"
OPENAI_API_BASE="http://localhost:1234/v1"
LLM_MODEL="local-model"
FLASK_DEBUG="False"
```

### Running the Simulator
```bash
python app.py
```
Open `http://localhost:5050` in your browser. Every debate turn is automatically appended to `storage/debates_dataset.jsonl` for your future training pipelines.

### Adding Custom Characters
Drop standard Character PNGs (TavernAI/SillyTavern format) into the `images/` folder, then run:
```bash
python extract_chunks.py
```
This extracts the hidden metadata and updates `characters.json`. (Note: The repository already includes Several dummy master debaters like Shikamaru Nara, Kyoko Kirigiri, Arnold Vinick, and Barack Obama to get you started!)

## 2. Setting up the RAG & Research Engine

### Installation
If you wish to run the background research tools, install the heavier dependency stack:
```bash
pip install -r requirements-rag.txt
```

### Running the RAG Server
```bash
python rag_server.py
```
The RAG server runs on Port 5080. If running, the Debate Simulator (`app.py`) will automatically detect it and inject relevant context into the characters' system prompts before they speak.
